syncmarx 0.1.0 (alpha)

Free, simple Web Extension for synchronizing bookmarks between browsers.

Syncmarx leverages cloud file hosting services to synchronize your bookmarks between browsers. Simply install the syncmarx extension, authenticate with a cloud file hosting service (currently limited to Dropbox), and create a local bookmark profile. Each bookmarks profile is saved to an individual file, and by setting up multiple profiles you can use the same cloud hosting account to store multiple sets of bookmark data.

Let the syncing begin!

Known Issues:
  • May not be able to reconcile duplicates within the same folder
  • Will not manage browser-specific bookmark functionality due to Web Extension spec limitations (e.g. Seperators, tags, keywords, description, favicons, etc.)
  • Will not track Firefox's "Other Bookmarks" folder (a.k.a. "Unorganized" bookmarks)
  • Still overall alpha in general so it is recommended to backup your bookmarks before using
